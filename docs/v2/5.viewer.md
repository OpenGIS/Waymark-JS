# Viewer Options

These options, provided in the `viewer_options` object, are used to customise the Viewer Mode.

> [!IMPORTANT]
> The following options are planned for a future release and are not yet implemented.

| Option             | Values  | Description                                                                      | Example             |
| ------------------ | ------- | -------------------------------------------------------------------------------- | ------------------- |
| `show_sidebar`     | `1`/`0` | Whether to show the sidebar.                                                     | `1`                 |
| `show_controls`    | `1`/`0` | Whether to show the map controls.                                                | `1`                 |
| `show_zoom_control`| `1`/`0` | Whether to show the zoom control.                                                | `1`                 |
| `show_location_control`| `1`/`0` | Whether to show the location control.                                            | `1`                 |
